# Консольный файловый менеджер

## Базовые классы приложения

Базовые классы:
1. Наследованные от абстрактного класса *FileDirectorySystemObject*:
- *DirectoryClass*;
- *FileClass*.

В данных классах инкапсулируется функционал базовой библиотеки использования файловой системы.

2. Наследованные от абстрактного класса *FileManagerCommand*:

3. Класс *UserParameters*. Хранит в себе последнее состояние консольного файлового менеджера. Сохраняет и загружает настройки. При их отсутствии загружает "дефолтные". Записывает ошибки, возникающие в ходе работы программы.
Настройки пользователя программа хранит в папке *Мои документы* в папке приложения. Там же находится и папка с отловленными ошибками файлового менеджера.

## Команды
help - выводит список команд и их описание

user - редактирование данных пользователя

cd - поменять текущую директорию

drives - вывести на экран список дисков

crdir - создать новую папку

crfile - создать файл

del - удалить папку или файл

move - переместить папку или файл

copy - копировать папку или файл

size - вычислить размер файла или папки

page - перейти на номер страницы

После ввода команды интерфейс пользователя спрашивает параметры, например указать новую страницу или путь копирования файла или директории
