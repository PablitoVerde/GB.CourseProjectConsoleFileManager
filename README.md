# Консольный файловый менеджер

## Базовые классы приложения

Базовые классы:
1. Наследованные от абстрактного класса *SystemObject*:
- *DirectoryClass*;
- *FileClass*.
В данных классах инкапсулируется функционал базовой библиотеки использования файловой системы.

2. Наследованные от абстрактного класса *FileManagerCommand*:
- *FileManagerCommandHelp*: выводит на экран список доступных команд.
- *FileManagerCommandEditUser*: выводит интерфейс редактирования пользовательских настроек.

3. Класс *UserParameters*. Хранит в себе последнее состояние консольного файлового менеджера. Сохраняет и загружает настройки. При их отсутствии загружает "дефолтные". Записывает ошибки, возникающие в ходе работы программы.

## Команды
help - выводит список команд и их описание
user - редактирование данных пользователя




* показывать содержимое дисков;
* создавать папки/файлы;
* удалять папки/файлы;
* переименовывать папки/файлы;
* копировать/переносить папки/файлы;
* вычислять размер папки/файла;
* производить поиск по маске (с поиском по подпапкам);
* для текстовых файлов готовить статические данные (кол-во слов, кол-во строк, кол-во абзацев, кол-во символов с пробелами, кол-во слов без пробелов).
